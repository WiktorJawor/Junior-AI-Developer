

Aplikacja napisana w Pythonie sluzaca do przeksztalcenia artykulu tekstoweo na strukturalny kod HTML za pomocą openAI API, dodaje tagi html (nagłówki, paragrafy listy, miejsca na zdjecie)

- konwert tesktu na kod HTML
- automatyczne dodane tagi <img>
- generuje podglad artykulu artykul.html
- generuje podglad z stylami css podglad.html
- genruje plik ze stylami szablon.html

Klucz do API został użyty przesłany przez Oxido należy go dodac w sekcji API_KEY

w pliku requirements znajduja sie wymagane pakiety
aby je zainstlować:
pip install -r requirements.txt

uruchomienie przez terminal:
python app.py
